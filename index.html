Conceptos de la clase


Desventajas de CSS Plano: Cuando usas CSS directamente sin un preprocesador como SASS o LESS, tiene sus cosillas negativas. Por ejemplo, no puedes usar variables, funciones o anidamiento, lo que hace que el código sea menos fácil de mantener y más propenso a errores. Además, en proyectos grandes y complejos puede ser un poco repetitivo y tedioso. Ir a google


Preprocesador: Un preprocesador es como una herramienta que amplía las capacidades del CSS normal, permitiéndote usar cosas como variables, funciones y otras características que luego se compilan en CSS normal antes de que el navegador las use. Esto hace que escribir CSS sea más eficiente y fácil de mantener.


SASS: es uno de esos preprocesadores de CSS que hace que tu vida como desarrollador sea más fácil y divertida. Introduce cosas como variables, anidamiento, mixins (que son como funciones reutilizables) y muchas otras cosas geniales. ¿Sabías que tiene dos formas de escribirlo? La forma más antigua, llamada simplemente SASS, y la forma más nueva, llamada SCSS, que es muy similar al CSS normal.


Compilación: Cuando trabajas con SASS, escribes tu código en archivos con extensión .scss, los cuales luego necesitas compilar a un archivo .css que el navegador pueda entender. La compilación es el proceso de convertir tu código SASS en CSS plano.


Las variables en SASS te permiten almacenar valores que deseas reutilizar en todo tu código. Por ejemplo, puedes definir un color como $first-color: black; y luego usar esa variable en lugar de tener que recordar el valor black en múltiples lugares.


Anidamiento: El anidamiento en SASS te permite escribir reglas CSS de manera más estructurada y legible al anidar selectores dentro de otros selectores, en este caso le cambiamos el tamaño de un a que este dentro de un p.

Modularidad: La modularidad en SASS se refiere a la capacidad de dividir tu código en archivos más pequeños y manejables, que luego puedes importar en tu archivo principal. Esto facilita la organización y reutilización del código. Por ejemplo creamos un archivo _otros.scss y lo importamos en styles.scss. _otros.scss es un archivo partial.
